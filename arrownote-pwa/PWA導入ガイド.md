# ArrowNote PWA 導入ガイド

## 1. アップロードする

`arrownote-pwa` フォルダーの中身を、HTTPSに対応したホスティングへそのままアップロードします。フォルダー構成やファイル名は変更しないでください。

初回テストでは、Cloudflare Pages、Netlify、GitHub Pagesなどの静的サイト向けサービスを利用できます。記録データはブラウザー内に保存されますが、URL自体を公開したくない場合はアクセス制限を設定できるサービスを選びます。

## 2. Safariで動作確認する

1. iPhoneのSafariで配置先URLを開きます。
2. 記録を1件保存し、履歴に表示されることを確認します。
3. Safariを閉じて再度開き、記録が残っていることを確認します。
4. 機内モードにして起動し、オフラインでも画面が開くことを確認します。

## 3. ホーム画面へ追加する

1. Safari下部の共有ボタンをタップします。
2. 「ホーム画面に追加」を選びます。
3. 名前が `ArrowNote` になっていることを確認して追加します。
4. ホーム画面のArrowNoteアイコンから起動します。

## 4. 更新するとき

同じURLへ新しいファイル一式を上書きします。反映されない場合は、ArrowNoteを完全に終了して再起動します。大きな更新では `sw.js` の `CACHE` 名を変更します。

## 5. データについて

記録は現在、そのiPhone内のブラウザーストレージに保存されます。アプリ削除、サイトデータ消去、端末故障に備え、定期的にNotion用CSVを出力してください。正式利用前にはJSON形式の完全バックアップ・復元機能を追加することを推奨します。

## テスト項目

- 日付・種目・着弾の入力
- 36本／72本の保存
- 履歴の全着弾表示
- 非採点練習のまとまり表示
- 名言の切り替え
- デザイン設定の保持
- Notion用CSVの出力
- オフライン起動
